# 1. TypeScript

## Beginner

* What is TypeScript and why use it instead of JavaScript?
* Difference between interface and type?
* Difference between any, unknown and never?
* What are generics?
* Explain enums.
* What are union and intersection types?
* Difference between optional and nullable?

## Intermediate

* How does TypeScript support dependency injection?
* Explain mapped types.
* Explain utility types (Pick, Omit, Partial, Required, Record).
* What are conditional types?
* What is declaration merging?
* Explain type narrowing.
* What are discriminated unions?

## Senior

* How would you design a reusable SDK using TypeScript?
* How do you enforce strict typing across frontend and backend?
* How do you type large REST responses?
* How do you organize types in a monorepo?
* How would you migrate a million-line JS codebase to TypeScript?
